# BASH

- `Shell`提供使用者对操作系统进行操作的一个接口，各个指令都是一个单独的应用程序
- 检测系统中的可以`Shell` 文件，在`/etc/shells`文件中
- 系统使用者默认取得的`Shell`，可以通过`/etc/passwd`进行查看
- 记录历史命令
  - `.bash_history` -- 记录上次登录所执行过的命令
  - 本地使用的命令记录在内存中，可以通过`.bash_history`查看上次登录操作
- 命令与文件补全功能 -- 使用`tab`案件
- 命令别名设置
  - `alias`
  - `alias lm='ls -al'`
- 工程控制，前景背景控制
- 程序化脚本
- 万用字符
